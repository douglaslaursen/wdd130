General Experience and Reflections

  • What was the most challenging part of this project for you? Were you able to overcome it?

    • The most challenging part was getting the CSS to look how I/my client wanted it to look. Luckily, I have a great TA and teacher to help me with questions. Thank you guys!

  • What did you enjoy most about this project?

    • With the most challenging, when CSS worked and making the page look nice! I like seeing the change of just HTML and then adding CSS to see what you can do! It is wild what CSS can do.

  • What would you do differently if you could do the project again?

    • Make sure I have more time to play with CSS. and think ahead of what IDs, Classes, Sections, and divs I want in the HTML to make making the CSS more smooth.

  • Do you feel more confident in working on real-world projects after this experience? Explain.

    • I feel way more confident then before, but if I was to get a job tomorrow and had to start making websites for them I would be really stressed. I would probly be able to do the basics and ask questions to make them happy, but I still have so much to learn for real-world applications.
